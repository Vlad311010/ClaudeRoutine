# TaskManager CLI

A simple command-line task manager built in C#. Demo project for showcasing Claude Code routines.

## Usage

```
dotnet run -- list
dotnet run -- list todo
dotnet run -- add "Fix login bug" "Users can't log in on Safari" High
dotnet run -- start 1
dotnet run -- done 1
dotnet run -- delete 2
dotnet run -- search "bug"
dotnet run -- stats
```
